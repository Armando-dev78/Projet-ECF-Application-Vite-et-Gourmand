# Vite & Gourmand

Application web développée dans le cadre de l’ECF Développeur Web & Web Mobile.

## Présentation

Vite & Gourmand est une application de gestion de commandes pour un traiteur événementiel basé à Bordeaux.

Elle permet :
- Consultation des menus
- Filtrage dynamique
- Création de compte
- Connexion sécurisée
- Passage de commande avec règles métier
- Calcul dynamique du prix
- Suivi des commandes utilisateur

---

## Stack technique

Front-end :
- HTML5
- CSS3
- JavaScript

Back-end :
- PHP 8 (architecture MVC simplifiée)
- PDO (requêtes préparées)

Base de données relationnelle :
- MySQL (phpMyAdmin)

Base de données NoSQL :
- MongoDB (prévue pour statistiques administrateur)

Serveur local :
- XAMPP
